# Netflix Data Analysis Dashboard

## Overview

This project is a data analysis dashboard built using Python and Jupyter Notebook.
It focuses on exploring and visualizing Netflix data to understand content trends, distribution, and insights.

## Features

* Analysis of Movies vs TV Shows
* Year-wise content growth
* Country-wise content distribution
* Genre/category insights
* Data cleaning and preprocessing
* Visualizations using charts and graphs

## Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Structure

* `Netflix_Dashboard.ipynb` → Main notebook
* `dataset.csv` → Dataset used for analysis
* `README.md` → Project documentation


## How to Run the Project

1. Download or clone the repository
2. Open the notebook in Jupyter Notebook
3. Run all cells step-by-step
4. View the visualizations and insights
